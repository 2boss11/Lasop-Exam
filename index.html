1. Explain the difference between forEach, map, filter, and reduce.

1. The forEach method doesn't return a new array, but it can be used to modify the original array. The map method returns a new array with the same length as the original array. The filter method returns a new array with only the elements that satisfy the provided condition. The reduce method returns a single value that is the accumulated result of the callback function.
